# 电源控制盒故障排查

:::info 端子位置参考
在进行故障排查前，建议参考 [**24V 电源控制盒说明书**](../../hardware-specifications/power-box/) 了解各个端子的位置和功能。
:::

## 按下软开关，不开机

检查事项，逐步做，如果一项

a. **确保空气开关打开** - 检查空气开关是否开启。如果已经开启，则继续下一步。
b. **确保不是软开关接触不良** - 拔下按钮线(最右侧6P)。用镊子把 左数 2-3 pin 导通。看能否开启。如果不能，继续下一步。
c. **确保电源有电** - 拔下电源盒的输入电源（最左侧最大的黄色端子），用万用表测量电压。正常电压是 20-30V。如果电压正常，插回去，继续下一步。
d. **确保没有外设短路** - 拔掉所有输出(2x4P, 2x3P，2x2P)，再用镊子试着开机（参见第二步）。如果还无法开机，确认是 power-box 需要更换。

## 按下软开关，立刻听到一声5秒的长鸣，然后自动关机

5秒长鸣，是触发了短路保护。排除办法：

a. 拔掉输出 2x4P、充电线，再次按软开关开机。如果依然 5秒长鸣，继续下一步。
b. 拔掉输出 2x3P, 2x2P，再尝试。如果依然长鸣，则确认是 power-box 需要更换。
