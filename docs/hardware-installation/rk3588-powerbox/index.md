# RK3588 算力盒 + 电源盒方案优势介绍

## 概述

新的算力平台基于 RK3588 研发，封装成两个部件：算力盒 + 电源盒，用以替换老一代平台（树莓派4B）。

![RK3588 算力盒与电源盒方案|600](./img/rk3588-overview.png)

## 主要功能性区别

| 功能/接口         | 老树莓派平台                | RK3588 + 电源盒新平台                      |
|------------------|---------------------------|------------------------------------------|
| 核心算力         | 树莓派4B                   | RK3588，CPU算力提升3-4倍，带7T NPU算力      |
| WIFI             | 树莓派自带WIFI，信号差      | 内置Intel AX200，支持WIFI6/企业级认证/双天线 |
| 蓝牙             | 蓝牙5.0                    | 蓝牙5.2                                   |
| 4G               | 无（需外接，易掉线）        | 内置4G模块，稳定性高                       |
| GNSS             | 无                         | 支持GPS和北斗                              |
| USB              | 2xTypeA, 2xMini，电流弱     | 4xTypeC, 2xTypeA，电流强，支持多相机        |
| 串口             | 3路                        | 4路                                       |
| CAN              | -                          | 2路，最大1M速率                            |
| 近场通讯         | 无（外接，不稳定）          | 内置，稳定性高                             |
| 以太网           | 1路1G                      | 2路1G                                   |
| 灯带控制         | 5V 2812b                   | 5V 2812b (4A电流)                         |
| 功放             | 无（需外接）                | 内置，驱动8欧双声道喇叭                    |
| 对外供电端子     | 2x24V, 1x12V, 1x5V         | 3x24V, 3x12V, 1x5V                        |
| 电源保护         | 熔断性保险丝                | 电子保险丝，短路无需修复                    |
| 封装             | 裸板                       | 电源盒+算力盒+支架                         |

## 其他改进

- 内置 IMU：内置 IMU，不容易出现接触不良的故障
- 关机功耗：新平台关机后MCU也断电，几乎无功耗
- 静电防护：新平台按钮、端子有光耦隔离和静电防护
- 对外供电：新平台有冗余供电接口
- 防感性反冲：新平台24V口有防感性器件反冲能力，80V耐压
- 电流采样：新平台有两路电流采样。顶升有单独的采样。
- 接口易用性：新平台主要接口侧面外露，无需拆主板
- 易维修性：新平台供电部分独立成盒，便于更换
